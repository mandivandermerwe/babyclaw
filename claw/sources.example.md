# News Sources

## Source Tiers

Organize your sources into tiers:

### Tier 1 — Primary
Your most trusted outlets. List them with URLs and notes on what they're good for.

### Tier 2 — Regional
Region-specific sources for deeper coverage.

### RSS Feeds
List verified RSS feed URLs (preferred for reliable parsing).

## Notes
- Document which sources have RSS, which require scraping.
- Note any access issues (paywalls, Cloudflare blocks).
- Specify User-Agent for HTTP requests.

See the private `babyclaw-preferences` repo for a working example with 11+ verified independent media RSS feeds across Iran, Middle East, Turkey, East Asia, and Global coverage.
